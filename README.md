## Description

[CoCode](https://github.com/parsasi/cocode) Development tutoring, revamped!

## Installation

```bash
$ npm install
$ cd /public npm install
$ npm run build
```

## Running the app

```bash
# development
$ npm run start
$ cd /public npm start


# watch mode
$ npm run start:dev
$ cd /public npm start


# production mode
$ cd /public npm install
$ npm run build
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
